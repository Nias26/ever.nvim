## Requirements
Neovim >= 0.10.0

ripgrep (rg)

git

## Installation
Clone this repo to your config folder
#### https
```
git clone https://github.com/Nias26/ever.nvim.git --depth 1 $HOME/.config/nvim
```
#### ssh
```
git clone git@github.com/Nias26/ever.nvim.git --depth 1 $HOME/.config/nvim
```
